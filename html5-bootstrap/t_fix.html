How to fix the FG-05 machine

This section gives you all the information you need to clean your ice cream machine

If the machine ain't broken, don't try to fix it.

  1. Open the machine by kicking its right side.

    Always unplug the ice cream machine before cleaning or servicing it.

  2. Determine the problem.
    • If the machine was making an ugly noise, then replace the flux capacitor.
    • If the machine was not working at all, then replace the power element.
  3. Evaluate the problem.
    ProblemSolution
    Ice cream is sour Clean the filter
    Ice cream is yummy Well, that's not a problem, Einstein!
